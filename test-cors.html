How to fix CORS issue:

  1. Install Node.js from https://nodejs.org/ (LTS version)
  2. Open a command prompt in your project folder
  3. Run: npx http-server -p 8080
  4. Open your browser and go to: http://localhost:8080/link.html

Alternative Solution (if you have Python installed):

  1. Open a command prompt in your project folder
  2. Run: python -m http.server 8000
  3. Open your browser and go to: http://localhost:8000/link.html
